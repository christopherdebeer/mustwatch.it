For Mobile (iOS)

If Safari's bookmarks sync with your iOS device, you should be able to go to your favorites and find the mustwatch.it! bookmark.

However, if that is not the case, you can add the bookmarklet onto your device by following this small guide.

  1. Select the code bellow and Copy all the text.
  2. Bookmark this page.
  3. Open your bookmarks.
  4. Press the Edit button on the lower, left-hand corner.
  5. Find and select mustwatch.it.
  6. On the Edit Bookmark page, delete the URL text.
  7. Once cleared, Paste the code.
  8. Press Done.
